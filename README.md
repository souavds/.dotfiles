# souavds' dotfiles

## Installation

```bash
git clone git@github.com:souavds/.dotfiles.git

# Choose a host and run its script
sh ./.scripts/host/arch.sh
sh ./.scripts/host/darwin.sh

# Or just stow the config files
stow -D . && stow .
```
